A page of contiguous memory, backed by a Buffer, representing objects in memory.

Every call to the region of virtual memory backed by a BufferPage goes straight through to the buffer underlying the page.

            |
            | get Field at position X
            V
      virtual buffer
            |
            | get Field at position X
            V
        BufferPage
            |
            | get Field at position X
            V
          Buffer
