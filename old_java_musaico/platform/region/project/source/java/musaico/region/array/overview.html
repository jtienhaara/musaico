Array regions, such as lists of objects, byte streams, and so on.

An ArrayRegion is described by a number of indices, typically starting at 0 and ending at (size - 1).

For example, an array of 32 objects can be described by an ArrayRegion with ArraySize ( 32 ) and start ArrayPosition ( 0 ) , followed by 1, 2, 3, ... and finally end ArrayPosition ( 31 ) .
